### 文件说明

文件集：

* profile：画像数据存放路径
* recallData：召回数据存放路径
* ranting_out：存放输出文件

.py文件：

* `datawash.py`：数据清洗文件
* `lr.py`：训练模型文件
* `trains_maker.py`：推荐文件
* `all_rating.py`：所有数据打分文件
* `final_rating.py`：预测数据打分文件

输出文件说明：

* `LR_32_1.h5, LR_32_2.h5, LR_64_1.h5, LR_64_2.h5`：训练好的模型，参数分别为一层32维度网络、两层32维度网络、一层64维度网络、两层64维度网络
* `rankings_20_all.csv`：打分结果
